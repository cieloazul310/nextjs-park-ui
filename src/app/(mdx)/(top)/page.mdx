## Features

- [Next.js v14 (App Router)][Next.js]
- [Park UI v0][Park UI]
- [Panda CSS v0][Panda CSS]
- [Storybook v7][Storybook]
- [MDX]
- Dark Mode by [next-themes v0][next-themes]

## Project structure

```txt
.
├── public
├── src
│   ├── app // Next.js App Router 
│   ├── components
│   ├── lib
│   ├── styles
│   ├── theme // for Panda config
│   ├── utils
│   └── mdx-components.tsx
├── next.config.mjs
├── package.json
├── panda.config.ts
├── park-ui.json
├── postcss.config.cjs
└── tsconfig.json
```
